# Luz da Lua 🍫

Sitio web de **Luz da Lua** — brownies artesanales en Guayaquil, Ecuador. Hechos a mano por Gigi con ingredientes reales.

## Qué incluye

- Catálogo de productos (brownies 1 y 2 pisos, shots, packs)
- Builder interactivo para armar tu brownie con base + toppings
- Carrito con integración directa a **WhatsApp**: al enviar el pedido, abre el chat del chef con el detalle completo (productos, cantidades, total) prellenado
- Diseño responsive (móvil, tablet, desktop)
- SVG animado de vista previa en vivo mientras personalizas

## Desarrollo local

```bash
python -m http.server 8000
```

Abre http://localhost:8000/

## Stack

HTML + CSS + JS vanilla. Sin dependencias, sin build. Despliegue directo a Vercel / GitHub Pages.
